At the 2017 Venice Biennale, artists were asked to select books for a fictional library as part of the "Unpacking My Library" project. This project, inspired by Walter Benjamin's essay about libraries, aimed to give the public insight into the artists' personal reading preferences and the books that influenced their work, ARCHIVIBE notes. The selected books were included in the official catalogue of the Biennale
